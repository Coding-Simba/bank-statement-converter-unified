Security Questions

Can your employees see my bank statements?

No. Our system is designed with zero-knowledge architecture. Employees cannot access uploaded files, even for support purposes. All processing is fully automated.

Do you keep logs of my transactions?

We only log technical metadata (file size, processing time) for system monitoring. We never log transaction details, amounts, or any financial information from your statements.

What if I accidentally upload the wrong file?

Files are automatically deleted after 60 minutes. You cannot retrieve files once uploaded. For immediate deletion, contact support with your session ID.

Is your service PCI compliant?

While we don't process payments, we follow PCI DSS security standards for handling sensitive financial data.

How do you ensure server security?

Our servers are hosted on AWS with:

  • Virtual Private Cloud (VPC) isolation
  • Web Application Firewall (WAF)
  • DDoS protection
  • Regular security patches and updates
  • Intrusion detection systems
