Integration Guide

To integrate this modal into your conversion process:

1. Include the modal script in your page:

<script src="/js/conversion-limit-modal.js"></script>

2. Call the modal when conversion limit is reached:

// For non-logged in users (after 2 conversions) showConversionLimitModal(false); // For logged in users (after 5 conversions) showConversionLimitModal(true);

3. Example implementation in your conversion handler:

function handleConversion() { const conversionsToday = getConversionsToday(); const isLoggedIn = checkUserLoginStatus(); const limit = isLoggedIn ? 5 : 2; if (conversionsToday >= limit) { showConversionLimitModal(isLoggedIn); return false; // Prevent conversion } // Continue with conversion... performConversion(); }
